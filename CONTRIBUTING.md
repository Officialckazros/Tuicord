# Contributing 🛠️💜

Oh hey, you want to help? That's genuinely adorable and I love you for it.

## The 30-second version
1. Fork it 🍴
2. Make a branch — `git checkout -b my-very-cool-change`
3. Do the thing ✨
4. Make sure it runs and doesn't burst into flames 🔥
5. Open a PR and tell me what you did (and why)

## House rules (gentle ones)
- **Keep PRs focused** — one idea per PR, not a 47-file mystery box 📦
- **Match the existing style** — when in Rome, semicolon like the Romans
- **No secrets** — don't commit tokens, `.env` files, or `node_modules` (I do not need your 4GB)
- **Be nice** — see the [Code of Conduct](CODE_OF_CONDUCT.md) (tl;dr: don't be a goblin 🧌)

## Scared of code but found a bug?
Totally fine! Just open an issue and describe the weirdness. Words are contributions too. 🗣️
